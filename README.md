# CAPTCHA Generator

This is a simple CAPTCHA generator built using HTML, CSS, and JavaScript.

## 🔗 Live Demo
https://your-username.github.io/captcha-generator/

## 📌 Features
- Generate random CAPTCHA text
- Refresh CAPTCHA
- Input validation
- Error message for wrong CAPTCHA

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Open the app
2. Enter the CAPTCHA shown
3. Click submit
4. Check if it is correct or not
